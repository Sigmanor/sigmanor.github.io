Установка и первый запуск

Архив с программой можно распаковать в любую удобную для вас директорию. Для удачного запуска потребуется Windows XP SP3 и выше. При первом запуске нужно выбрать язык программы, потом откроется окно с Лицензионным соглашением, для дальнейшей работы необходимо нажать кнопку принять. Это стандартная MIT лицензия, с русским переводом можно ознакомиться например тут.

Внимание! Лаунчер не умеет скачивать или обновлять игру.

Скриншоты

Подготовка к работе

При первом запуске, путь к игре будет найден автоматически (при наличии установленного официального ланучера). Если по какой то причине этого не случилось, то необходимо зайти в Меню > Настройки в пункте Папка с игрой нажать на кнопку с изображением папки, и выбрать папку с игрой. После этого лаунчер готов к запуску игры.

Управление несколькими аккаунтами

Для удобства, что бы каждый раз не вводить логин/пароль, можно ввести их в главном окне лаунчера, и поставить галочку на Сохранить. Так же есть возможность сохранить несколько аккаунтов, для этого в поле ввода логина в выпадающем списке нужно выбрать пустую область, и ввести в поля данные второго аккаунта, поставить галочку на Сохранить, теперь из выпадающего списка можно выбрать несколько аккаунтов для игры. Таким способом можно добавить бесконечное количество аккаунтов. Так же можно редактировать уже существующие данные, для этого из выпадающего списка нужно выбрать нужный аккаунт, отредактировать данные в полях, и поставте галочку на Сохранить. Удаление аккаунтов из лаунчера на данный момент не возможно. Волнует безопасность данной функции?

Дополнительно

Для того что бы узнать что скрывается в поле пароля, нужно справа нажать на кнопку с изображением глаза. Для того что бы узнать больше информации о программе, или вручную обновить, необходимо зайти в Меню > Помощь. Так же, в Меню > Дополнительно можно запустить официальный лаунчер. Слайдер в главном окне взят из официального лаунчера, с помощью кнопок ниже можно перейти на интересные ресурсы посвященные Американскому серверу Aion.

Настройки лаунчера

Общие

Папка с игрой: Выбор папки с установленной игрой, нужно выбирать корневую папку.

Доп. Параметры: Поле для ввода дополнительных параметров запуска игры, указывать через пробел. Некоторые параметры с описанием:

  • -ncping включает функцию NCPing в настройках игры.
  • -megaphone включает мировой чат.
  • -nowebshop отключает отображение иконки внутриигрового магазина.

Разрядность: Выбор разрядности игрового клиента.

Язык лаунчера: Выбор языка программы.

Игровой сервер: Выбор сервера, статус которого будет отображаться в главном окне программы.

Расширенные

Файл параметров: В поле отображен путь к файлу настроек лаунчера, который создается автоматически при первом запуске. Кнопка справа позволяет открыть в проводнике папку с файлом.

Сайт мониторинга: Выбор с какого сайта получать статус сервера.

После запуска: Позволяет выбрать поведение лаунчера после запуска игры.

Быстрый запуск: При включении, игра будет запускается через 3 секунды после запуска лаунчера. Лаунчер запускается свернутым, по этому если понадобиться остановить отсчет, можно развернуть его, и в строке уведомлений будет отсчет до запуска, нажатием на кнопку можно остановить/возобновить таймер.

Авто. Проверка обновлений: При включении, обновления лаунчера будут проверятся автоматически, и в главном окне, в строке уведомлений, будет написано оповещение о новой версии.

Предупреждение о рестарте: Включение функции предупреждения о рестарте, которое будет отображаться в строке уведомлений в главном окне.

Показывать пинг: Включение функции отображения пинга.

Часто задаваемые вопросы

В: Меня волнует безопасность введенных данных от игрового аккаунта в лаунчере.
О: Ваши данные никому не передаются, программа полностью безопасна. В этом можно лично убедиться, так как исходные коды находятся в открытом доступе на GitHub.

В: Почему в лаунчере показывает что сервер работает, но в игру не заходит.
О: Информация берется из сторонних сайтов: aion.im, aionstatus.net, aion.mouseclic.com, смотря что выбрано в настройках, по этому все вопросы к разработчикам данных ресурсов.

В: Почему пинг в лаунчере отличаеться от пинга в игре?
О: В лаунчере идет проверка пинга к IP адресу сервера через командную строку напрямую, и многие маршруты не учитываются.

В: У меня не запускается игра/лаунчер/показывает какую то ошибку.
О: Для начала нужно попробовать запустить игру через официальный лаунчер, если ошибка останется, то проблемы не в лаунчере, если же ошибки нет, то попробуйте следующие решения:

1) выбрать в настройках другую разрядность;

2) убедиться что ваша система соответствует системным требованиям;

3) попробовать сбросить настройки лаунчера на исходные;

4) если ничего не помогло, то напишите мне (контакты есть в шапке сайта), приложив скриншот ошибки.
